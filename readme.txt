== Description ==

Random Quote plugin by KnowKwote.com. Displays a random quote in a widget, pulled from the KnowKwote servers.

== Installation ==

1. Download the Plugin from either the WordPress Plugin Directory or from within your WordPress admin panel.

2. Install the Plugin by clicking "activate" and then proceed to the "widgets" panel to add the KnowKwote widget to your site.

3. Within the widget panel, you can adjust the KnowKwote widget settings.

== Frequently Asked Questions ==

= Can I customize the text? =

The plugin currently allows for customizing the text for bold, italics or regular text.
